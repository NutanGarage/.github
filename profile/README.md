# 🔧 Welcome to NutanGarage!

> *Where learning gets built, one repo at a time.*

Hey there! I'm **Nutan**, and this is my open-source learning garage — a place where developers at every level can find roadmaps, challenges, tutorials, and beginner-friendly projects to level up their skills.

Whether you're just starting out or sharpening your craft, there's something here for you. Pull up a chair, grab your tools, and let's build together. 🚀

---

## 🗂️ What's Inside?

| Repo | Description |
|------|-------------|
| 📍 `roadmaps` | Curated learning paths for web, backend, DSA & more |
| 💡 `coding-challenges` | Daily & weekly problems with detailed solutions |
| 📚 `resources` | Handpicked books, links & tools organized by topic |
| 🛠️ `projects-for-beginners` | Starter project ideas with step-by-step guides |

---

## 🚀 Our Mission

To make learning **open, accessible, and community-driven** — so every developer has a clear path forward, no matter where they're starting from.

---

## 🤝 Contributing

We welcome contributions from everyone!

1. Fork the repo you want to contribute to
2. Create a new branch: `git checkout -b your-feature`
3. Make your changes and commit: `git commit -m "Add: your change"`
4. Push and open a Pull Request

Please read the `CONTRIBUTING.md` in each repo before submitting.
